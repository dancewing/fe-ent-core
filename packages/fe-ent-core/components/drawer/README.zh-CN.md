## API


### `<ent-drawer>` Props

|参数名|描述|类型|默认值|模块|
|---|---|---|:---:|---|
|is-detail|是否明细页面|`boolean`|`false`|`-`|
|loading-text|装载中显示文本|`string`|`-`|`-`|
|show-detail-back|是否显示退回按钮|`boolean`|`true`|`-`|
|loading|是否正在装载|`boolean`|`false`|`-`|
|close-func|关闭后回调函数|`any`|`null`|`-`|
